Project Overview

Welcome to the Blockhouse Market Impact & Optimal Allocation Project! This initiative focuses on advanced quantitative analysis to model market impact and optimize trading strategies. Our goal is to minimize trading costs by understanding and predicting the temporary price impact of large orders and formulating an optimal execution algorithm.

Key aspects of this project include:

  • Goal: Minimize trading impact by modeling and optimizing order execution across multiple stocks and time periods.
  • Data: Utilizes real and synthetic order book data for 3 stocks (CRWV, FROG, SOUN) at 1-minute intervals.
  • Deliverables: Provides an interactive analysis report, professional plots, and detailed written explanations in PDF format.

All results, code, and documentation are provided below for easy access and review.
